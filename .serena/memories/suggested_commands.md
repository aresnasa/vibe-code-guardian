# Development Commands

## Building and Development
- `npm install` - Install dependencies
- `npm run watch` or `npm run compile` - Build TypeScript
- `npm run esbuild` - Bundle with esbuild
- `F5` in VS Code - Launch Extension Development Host

## Testing
- `npm test` - Run tests
- `npm run watch-tests` - Watch mode for tests

## Code Quality
- `npm run lint` - Run ESLint
- `npm run format` - Format code

## Packaging
- `npm run package` - Create .vsix file for distribution
